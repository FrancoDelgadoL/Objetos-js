# JAVASCRIPT

## OBJETOS

1. Creacion de objetos : Con let y const
2. Acceder a los valores: console.log(person.name) nos permite acceder a una propiedad de un objeto 
3. Agregar valores: Con el nombredelobjeto.valor 
4. Modificar valores: Con el nombredelobjeto.propiedad y se asigna un nuevo valor 
5. Eliminar valores: Con delete podemos borrar propiedades de un objeto 
6. Congelar: Con freeze nos bloquea el poder modificar, agregar o eliminar propiedades 
7. Sellar: Con Seal nos permite modificar mas no agregar o eliminar
8. Unir objetos: Nos permite unir dos o mas propiedades de un objeto y mostrarlos en la consola
9. Palabra reservada this en objeto literal: Creando una funcion, con la palabra reservada this, podemos retornat mas de una propiedad de un objeto.
10. Recorrer: Con un bucle for, podemos recorrer y mostrar todos las propiedades dentro de un objeto.
